🔗 Menu 提取工具

從 Foodpanda 頁面提取完整 menu（含冰度/甜度 options）

📝 方法一：貼 JSON（最簡單）

1
喺瀏覽器開 Foodpanda 餐廳頁，按 F12 開 Console
貼呢段 code 按 Enter：
copy(JSON.stringify(JSON.parse(document.getElementById('__PRELOADED_STATE__')?.textContent||'{}')?.vendor?.data?.menus?.[0]?.menuCategories?.flatMap(c=>c.products?.map(p=>({name:p.name,price:''+Math.round((p.minimumPrice??p.price??0)*100)/100,desc:p.description||'',hasOpts:(p.modifierGroups||[]).length>0}))||[])?.filter(x=>x.name)||[])) 自動複製 menu JSON 到剪貼簿
2
返嚟呢度，喺下面貼上（Ctrl+V），然後按「提取」
